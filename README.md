# Quick Start

[Analogweb](http://analogweb.org) is a tiny HTTP oriented web framework.

Installing [Scala](http://www.scala-lang.org/) and [sbt](http://www.scala-sbt.org) then run.

```
$ sbt run 
```

And you will get

```
$ curl localhost:8000/ping
$ PONG!
```
